# LLQQNT-Plugin-Template-TS

LiteLoaderQQNT的非官方TypeScript插件模板  
LiteLoaderQQNT本体：[LiteLoaderQQNT](https://github.com/mo-jinran/LiteLoaderQQNT)

## 命令
```bash
# 生产环境打包
npm run build
pnpm run build
yarn build

# 开发环境构建
npm run build:dev
pnpm run build:dev
yarn build:dev

# 开发环境构建后手动打包
npm run build:zip
pnpm run build:zip
yarn build:zip

# 清理 dist 目录
npm run clean
pnpm run clean
yarn clean
```
